Education Platform v1.0.0 - Release Notes
==========================================

🎉 INITIAL RELEASE - Production Ready

System Requirements:
- Java 17 or higher
- Windows, macOS, or Linux
- 512MB RAM minimum (2GB recommended)

QUICK START:
============

Windows:
  1. Download: education-platform-v1.0.0.jar
  2. Double-click run.bat OR execute:
     java -jar education-platform-v1.0.0.jar

Linux/Mac:
  1. Download: education-platform-v1.0.0.jar
  2. Run: ./run.sh OR java -jar education-platform-v1.0.0.jar

FEATURES INCLUDED:
==================

✅ User Authentication
   - Secure login/registration
   - SHA-256 password hashing
   - Role-based access (Student, Instructor, Admin)

✅ Quiz System
   - Create and take quizzes
   - Multiple question types
   - Automatic scoring and feedback

✅ Discussion Forums
   - Create forum threads
   - Post messages and reply
   - Engage with community through likes

✅ Video Lessons
   - Upload and manage videos
   - Track view counts
   - Download support

✅ Database
   - SQLite for data persistence
   - Automatic initialization
   - All data saved locally

INSTALLATION:
==============

1. Ensure Java 17+ is installed:
   java -version

2. Download the JAR file

3. Run using your preferred method:
   - Windows: double-click run.bat
   - Linux: bash run.sh
   - macOS: bash run.command
   - Any OS: java -jar education-platform-v1.0.0.jar

FIRST TIME SETUP:
=================

1. Application starts with login screen
2. Register a new account:
   - Choose role: Student, Instructor, or Admin
   - Set email and password
3. Login with your credentials
4. Start exploring!
